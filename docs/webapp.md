# Webapp

## Role

Webapp is a complementary browser surface.

It is useful for lighter inspection and backlog visibility, but it is not the
source-of-truth workflow for ingest, analysis control, or candidate review in
V1.

## Responsibilities

- project browsing
- candidate history viewing
- profile inspection
- settings/config visibility
- future collaboration/review workflows

## Current State

The scaffold includes placeholder screens for:

- Dashboard
- Projects
- Candidate History
- Profiles
- Settings

Desktop remains the primary review workstation.
