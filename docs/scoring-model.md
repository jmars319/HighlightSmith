# Scoring Model

## Signal Tiers

- Trigger signals
  - loudness spike
  - laughter burst
  - overlap spike
  - reaction phrase
  - action audio cluster
- Supporting signals
  - commentary density
  - tactical narration
  - pitch excursion
- Context signals
  - setup
  - consequence
  - resolution
  - silence break
  - abrupt silence after intensity
- Negative signals
  - menu heavy
  - cleanup heavy
  - low information
  - context required

## Confidence Model

The UI surfaces confidence bands:

- `HIGH`
- `MEDIUM`
- `LOW`
- `EXPERIMENTAL`

The system does not claim objective “best moments.” It surfaces likely moments worth reviewing.
